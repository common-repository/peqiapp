== Description ==

This plugin relies on an external API service, [https://api.peqi.app](https://api.peqi.app), to enhance its functionality. The plugin integrates with the service under circumstances related to account management, domain handling, and related functionalities.

For more details regarding the terms and conditions of the service, please refer to: [https://get.peqi.app/terms-conditions/](https://get.peqi.app/terms-conditions/)

== Installation ==

= Minimum Requirements =

* WordPress 4.7 or greater
* PHP version 7.0 or greater
* MySQL version 5.0 or greater

= Recommended Requirements =

* PHP version 8.1 or greater
* MySQL version 5.6 or greater
* WordPress Memory limit of 64 MB or greater (128 MB or higher is preferred)

= Installation =

1. Install using the WordPress built-in Plugin installer, or Extract the zip file and drop the contents in the `wp-content/plugins/` directory of your WordPress installation.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to Pages > Add New
4. Press the 'Start setup' button.

For documentation and tutorials visit our [Knowledge Base](https://help.peqi.app).

== Frequently Asked Questions ==

**How do I install Peqi?**

To install the Peqi, follow the steps below:
From your WordPress dashboard -> Go to Plugins -> Click on 'Add new'-> In the Search field, enter peqi and choose Peqiapp.
Press install -> After installation, click Activate.
